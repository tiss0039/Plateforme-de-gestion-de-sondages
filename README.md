# Plateforme de gestion de sondages

Cette application permet de créer, gérer et répondre à des sondages via une plateforme web et une application mobile.

## Fonctionnalités

- Création et gestion de sondages
- Gestion des questions
- Collecte des réponses
- Gestion des utilisateurs et des rôles
- Tableau de bord selon le rôle de l'utilisateur

## Technologies

- Laravel
- PHP
- MySQL
- Blade / Bootstrap
- Android
- API REST

## Installation

1. Cloner le projet

git clone https://github.com/tiss0039/Plateforme-de-gestion-de-sondages.git

2. Installer les dépendances

composer install

3. Copier le fichier environnement

cp .env.example .env

4. Générer la clé Laravel

php artisan key:generate

5. Lancer le serveur

php artisan serve
